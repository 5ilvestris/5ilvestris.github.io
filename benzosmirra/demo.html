API Anahtarları
⚠️ Tarama yapabilmek için en az bir API anahtarı gereklidir.
Anahtarlar backend/.env dosyasına kaydedilir ve yalnızca kendi cihazında çalışır.
🔍
SerpAPI ÖNERİLEN
Google Ters Görsel Arama — Ayda 100 ücretsiz
Ücretsiz al →
serpapi.com → Dashboard → Your Private API Key
🔬
FaceCheck.ID YÜZ TANIMA
Sosyal medya odaklı yüz arama — Token bazlı
API al →
facecheck.id → Sign In → API Token
👁️
TinEye TERS GÖRSEL
Ters görsel arama motoru — Ayda 100 ücretsiz
API al →
Format: api_key:api_secret (iki değeri iki nokta ile ayır)
🪟
Bing Visual Search
Microsoft Azure — Ayda 1000 ücretsiz işlem
Azure →
Azure Portal → Bing Search v7 → Keys and Endpoint → Key 1
💎
PimEyes ÜCRETLİ
En kapsamlı yüz arama motoru — Enterprise
API al →
pimeyes.com → Account → API Access → Bearer Token
🌐
Google Vision API 1000/AY ÜCRETSİZ
Web Detection — görsel üzerinden internet taraması
GCP →
Google Cloud Console → Vision API → Credentials → API Key
🖼️
ImgBB ÜCRETSİZ
Görsel hosting — URL gerektiren API'ler için gerekli
Ücretsiz al →
imgbb.com → Account → API → Add API key
📸
Imgur Client ID ÜCRETSİZ
Görsel hosting alternatifi — ImgBB yoksa kullanılır
API al →
imgur.com → Register Application → Client ID
